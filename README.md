# flight-delays-ml-project
Analysis and prediction of US flight delays using the 2015 DOT dataset.

## Datasets
Les datasets utilisés dans ce projet sont issus du **U.S. Department of Transportation (2015)**.

Pour des raisons de taille, les fichiers CSV ne sont pas inclus dans le dépôt GitHub.

### Fichiers attendus
Placez les fichiers suivants dans le dossier `data/` :
- `flights.csv`
- `airlines.csv`
- `airports.csv`
